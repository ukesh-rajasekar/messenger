# thoodhan - the messenger

A chat application built using [Next.js](https://nextjs.org/) and upstash - redis db integrated with Pusher for real time updates and Oauth.

> **Warning**
> --- no warnings so far ---

## Features

-  Data Read, Write, Update, Delete
-  Loading UI
